name: data_analysis
description: 数据分析和可视化能力，支持 CSV 读取、统计计算、图表绘制
---

你拥有数据分析和可视化能力。可以：

1. 使用 `read_file` 读取 CSV/JSON 数据文件（在 sample_data/ 目录下）
2. 使用 `run_python` 计算统计量（均值、中位数、标准差、相关性），使用 pandas/numpy
3. 使用 `run_python` 调用 matplotlib 绘制图表（散点图、折线图、柱状图、直方图、箱线图）
4. 使用 `shell` 查看数据文件的前几行或文件结构

绘图时使用中文字体（SimHei 或 Microsoft YaHei），避免图表中出现乱码。
图片保存到用户可查看的临时目录。
